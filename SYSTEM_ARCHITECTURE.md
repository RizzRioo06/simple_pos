## User Roles & Permissions

| Role     | Can View | Can Modify |
|----------|----------|------------|
| Customer | Own orders, Menu | Create orders, Submit payment |
| Kitchen  | All pending orders | Update cooking status |
| Waiter   | Ready orders, All tables | Serve food, Clean tables |
| Cashier  | Served orders | Verify payments |
| Admin    | All tables, All orders | All operations |

## Real-time Updates

All interfaces auto-refresh every 3 seconds via polling:
- Customer sees order status changes
- Kitchen sees new orders
- Waiter sees ready-to-serve items
- Cashier sees payment submissions
